# MyWeather App

Simple weather website built with plain HTML, CSS, and JavaScript.

## Features

- City search by name
- Browser geolocation auto-detection
- Current weather conditions
- 5-day forecast
- Language switcher (English / German)
- Responsive layout for desktop and mobile
- No backend required

## Data source

- Geocoding + weather API from Open-Meteo
- Website: https://open-meteo.com/

## Run locally

1. Open `index.html` in your browser.
2. Search for any city.

If your browser blocks some requests on local files, run a simple local server:

```bash
python3 -m http.server 8000
```

Then open:

`http://localhost:8000`
